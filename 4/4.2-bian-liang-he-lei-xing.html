4.2 变量和类型

4.2 变量和类型

  1. 尝试初始化声明它们的变量。
  2. 始终选择所需的最简单的数据类型，list 或 object。
  3. 始终使用内置的 C＃ 数据类型别名，而不是 .NET 公共类型系统（CTS）。
    Example：
     使用 short 而不是 System.Int16
 使用 int 而不是 System.Int32
 使用 long 而不是 System.Int64
 使用 string 而不是 System.String
  4. 仅将成员变量声明为 private。属性则使用 public，protected，或 internal 访问修饰符。
  5. 尝试对任何适合 int 数据类型的非小数数值使用 int - 甚至非变量负数。
  6. 仅对可能大于 int 取值范围的变量使用 long。
  7. 尝试使用 double 作为小数，以确保计算的小数精度。
  8. 仅对不适合 double 或 decimal 的小数 使用 float。
  9. 除非您完全理解对任何计算的影响，否则避免使用 float。
  10. 当小数必须四舍五入到计算的固定精度时，尝试使用 decimal。 通常是涉及金钱。
  11. 避免使用 sbyte，short，uint 和 ulong，除非它是与本机库的互操作（P/Invoke）。
  12. 避免指定枚举 enum 的类型 - 使用默认值 int，除非你明确需要 long（非常 罕见）。
  13. 避免使用内联数字（幻数）。 相反，使用常量 Constant 或枚举 enum。
  14. 避免内联声明字符串文字。 而是使用资源文件，常量，配置文件，注册表或其他数据源。
  15. 为复杂类型声明只读 readonly 或静态只读 static readonly 变量，而不是常量。
  16. 仅声明简单类型的常量 constants。

  17. 避免直接转换。 而是使用 "as" 运算符并检查 null。
    Example： 

    object dataObject = LoadData();
DataSet ds = dataObject as DataSet;

if(ds != null)
{...}
  18. 相对于标准或强类型集合，总是更倾向于 C# Generic 集合类型 [C#V2 +]
  19. 始终使用 for 循环显式初始化引用类型数组。
  20. 避免装箱 boxing 和拆箱 unboxing 值类型。
    Example： 
    int count = 1; 
object refCount = count; //隐式装箱 Implicitly boxed. 
int newCount =（ int ）refCount ; //显示拆箱 Explicitly unboxed.
  21. 浮点值应至少包含小数位前一位数和后一位数。
    Example：totalPercent = 0.05;
  22. 尝试使用 "@" 字符前缀代替转义字符。
  23. 在字符串拼接上倾向于使用 String.Format() 或 StringBuilder。
  24. 永远不要在循环内拼接字符串。
  25. 不要通过 String.Empty 或 "" 来比对检查字符串是否为空，相反，使用 String.Lenght == 0 会更好。

    译者注：新版本 C# 中建议使用 String.IsNullOrEmpty(String)

  26. 避免在循环中给字符串赋值。对区分大小写使用 String.Compare()
    Example：（ToLower() 创建一个临时字符串）

    // Bad!
int id = -1;
string name = “lance hunt”;

for(int i=0; i < customerList.Count; i++) 
{ 
    if(customerList[i].Name.ToLower() ToLower() == name) 
    { 
        id = customerList[i].ID; 
    } 
}

// Good!
int id = -1; 
string name = “lance hunt”;

for(int i=0; i < customerList.Count; i++) 
{ 
    // The “ignoreCase = true” argument performs a 
    // case-insensitive compare without new allocation.
    if(String.CompareString.Compare(customerList[i].Name, String.Compare name, true)==true 0) 
    { 
        id = customerList[i].ID; 
    }
