1.4.2 编码风格

1.4.2 编码风格

Code Style
代码文件 一个命名空间对应一个文件和一个类对应一个文件
大括号 总是新起一行
缩进 使用制表符或四个空格符
注释 使用 // 或 ///，但是不要使用 /* ... */
变量 一个变量只声明一次
